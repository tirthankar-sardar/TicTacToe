# Tic Tac Toe Game using Python

## Project Description
This project is a simple Tic Tac Toe game developed using Python in Google Colab. The game allows two players to play against each other by taking turns marking spaces on a 3×3 grid. It includes game logic for checking winners, validating moves, and detecting draws.

The project is beginner-friendly and demonstrates the use of Python programming concepts such as loops, functions, conditionals, and lists.

---

## Features
- Two-player gameplay
- Interactive command-line interface
- Winner detection
- Draw detection
- Simple and clean Python implementation

---

## Technologies Used
- Python 3
- Google Colab

---

## How to Run
1. Open the notebook in Google Colab.
2. Run all the cells.
3. Enter positions (1-9) when prompted.
4. Continue playing until a player wins or the game ends in a draw.

---

## Game Rules
- The game is played on a 3×3 board.
- Two players take turns:
  - Player X
  - Player O
- The first player to place three marks in a row, column, or diagonal wins.
- If all cells are filled and no player wins, the game ends in a draw.
